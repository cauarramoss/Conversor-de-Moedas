# Conversor de Moedas do DevClub

OBS: A ideia deste conversor eu vi em um vídeo do DevClub, e resolvi fazer para estudar e aprimorar minhas técnicas.

Um conversor de moedas simples e moderno, feito em HTML, CSS e JavaScript, que utiliza uma API pública para obter taxas de câmbio em tempo real.

## Funcionalidades

- Conversão entre as principais moedas do mundo (USD, EUR, BRL, GBP, JPY, CAD, AUD, CHF, CNY, ARS)
- Interface responsiva e amigável
- Taxas de câmbio atualizadas em tempo real
- Exibição clara do valor convertido e da taxa utilizada
- Mensagens de carregamento e erro para melhor experiência do usuário

## Link para usar o conversor

- conversordemoedas-caua.netlify.app

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- [ExchangeRate-API](https://www.exchangerate-api.com/) para taxas de câmbio

## Estrutura do projeto

- `index.html` — Página principal do conversor
- `style.css` — Estilos do projeto
- `script.js` — Lógica de conversão e integração com a API
